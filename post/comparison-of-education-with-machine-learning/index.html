Comparison of education with machine learning

2018-06-21

Let me speculate a little, and compare what does an educator (teacher/professor/instructor) with what does a machine learning specialist.

What is common?

  • Provide information
    • Educator gives lectures, provides a textbook, or other resources.
    • ML specialist have to find and present data
  • Adjusts information
    • Educator have to choose a suitable form and complexity level of information for a particular group of students
    • ML specialist have to clean and reshape data to a form an algorithm can understand
  • Give feedback
    • Educator have to correct and redirect efforts of a student when it is needed. Educators use example problems (when a solution is given) and practice problems (on which students test patterns learned from examples).
    • ML specialist observes how algorithm learns. If this process is inefficient, learning strategy shall be corrected. Training and validation (cross-validation) sets are used.
  • Perform assessment
    • Educator gives tests and makes exams, designed to see how a student performs on problems, which are similar to those covered in class, but at the same time different - such that student did not see before.
    • ML specialist evaluates the performance of the algorithm on a test set.
  • Focus on self-learning
    • Educator creates conditions for self-learning. A student is the only person who can actually learn. As stated by Isaac Asimov: “Self-education is the only kind of education there is”
    • ML specialist create conditions and then, in essence, wait for a machine to learn on its own
  • Work with models of reality
    • Educator aims to help students to develop a deep internal understanding (meaning, that at the very least they can use it in practice) of models of reality
    • ML specialist makes a computer create a model describing patterns in data
  • Use pre-trained models
    • ML specialist can use a pre-trained neural network and only moderately retrain it for a particular application
    • Educator provides methods (for problem-solving, research, etc.) which proven to be successful in specific areas. Students work on absorbing those and modifying them for themselves

What is different?

Where a machine excels

  • ideal information retention
  • fast extraction of unknown patterns, when learning task is clear
  • fast application of learned patterns
  • simple transfer of learned models from computer to computer

Where a human excels

  • learning from very few examples, efficient generalization
  • generalization to ‘data from new distributions’
  • setting up new tasks independently
  • challenging assumptions, and finding limits of applicability of various models
  • learning something as complicated as physics. Look at this interesting plot describing what infants do in their first year of life

Final remark

A question: Can we formulate education process using machine learning terminology? What if this will provide us with a more structured way to describe an educational process? Also, maybe we can bring something new to education? For example, what would be an analogy for a regularization technique used in machine learning?
