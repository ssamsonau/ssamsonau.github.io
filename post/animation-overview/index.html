Animation Overview

2017-11-22

Contents

A short review of some methods allowing to create animations in R is presented bellow.

Available options

  • library(animation)
    • saveGIF (relies on ImageMagick or GraphicsMagick)
    • saveVideo
    • saveMovie
    • saveHTML
    • other functions here
  • gganimate
    • additional use of package tweenr makes a result smother
  • animated maps

saveGIF

saveGIF requires to for ImageMagick or GraphicsMagick to be installed.

  • ImageMagick (how to install on MAC)
  • GraphicsMagick (how ot install GraphicsMagick on Windows). In saveGIF function, you have to use option convert = ‘gm convert’
  • alternativelly, one can use installr::install.ImageMagick()
if(!dir.exists(paste0(prefix, "/movies"))) 
  dir.create(paste0(prefix, "/movies"))
library(animation)
library(ggplot2)
  
saveGIF({
    for (i in 1:10) 
      plot(runif(10), ylim = 0:1)
        }, 
        convert = 'gm convert', 
        movie.name = paste0(prefix, "/test1.gif"))
## [1] TRUE

Brownian motion example from animation::brownian.motion

saveGIF({
  brownian.motion(
    pch = 21,
    cex = 5,
    col = "red",
    bg = "yellow"
  )
},
movie.name = paste0(prefix, "/movies/brownian_motion2.gif"),
interval = runif(30, 0.01, 1),
nmax = 30, convert = 'gm convert')
## [1] TRUE

Note: Number of steps is controled by nmax in saveGIF. Alternativelly, you change this globally with ani.options(“nmax” = 50)
