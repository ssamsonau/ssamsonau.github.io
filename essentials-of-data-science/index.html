  • DS-Essentials

There is a lot of information on Data Science on Internet. On this page I summarize the most essential things you would need to know to start working on real projects. The rest you can figure out on the way. This page is intentionally short (relativelly).

Books

  • R for Data Science
  • An Introduction to Statistical Learning
  • Data Science for Business

Tools, Installation and updating

  • installr package
    • for updating R to a newer version (from RGui do installr::updateR())
    • installing Rtools, git, ImageMagick, etc. Link
  • devtools package for installation of packages from github repositories directly

Interactive dashboards

  • R Shiny
  • R Shiny Gallery

Cheatsheets

  • dplyr and tidyr, ggplot, shiny sheet 1, shiny sheet 2, regular expressions

Databases

  • sqlzoo, sql

Parallel processing in R

  • Microsoft R Open & MKL - seamless use of more than one cpu core

Working with medium data

  • If data can still fit in RAM of local machine, but slow to process with dplyr, a package data.table can provide a faster alternative

Working with Big data (located in cluster)

  • sparklyr: R interface for Apache Spark
  • rsparkling: Sparkling Water (H2O) Machine Learning, including deep learning
  • R with Hadoop
  • Visualize Large Data

Text mining in R

  • tm package
  • Text Mining with R (book)

Plotting libraries

  • ggplot2 - see R for Data Science. There are many extentions for ggplot2. Some of them are here:
    • gridExtra to allign plots
    • ggbio - can be usefull in order to allign several plots (instead of gridExtra)
    • ggmap to make plots with geographical maps
    • ggpubr - modified ggplot2 for publication ready plots
  • plotly - makes ggplot2 interactive
  • ggvis - interactive plots (behind the scenes, interactive plots are built with shiny)
  • dygraph - R interface to the dygraphs JavaScript charting library. It provides rich facilities for charting time-series data in R
  • Visualize Large Data

Books with Rmarkdown

  • https://bookdown.org/yihui/bookdown/

Websites with Rmarkdown

Static sites (no need of databases, use of wordpress, etc.). Host for free on GitHub pages

  • https://www.rstudio.com/resources/webinars/introducing-blogdown/
  • https://slides.yihui.name/2017-rstudio-webinar-blogdown-Yihui-Xie.html
  • themes for hugo (a static site generator used by blogdown package)
    • find the one you like on this site https://themes.gohugo.io/
    • press Download - it should open a github address (if it is not GitHub it is ok..)
    • use this to install blogdown::install_theme(“user/repo”). It it is not on GitHub speicify hostname
    • fine tune the theme
    • host on GitHub pages or better in netfly

Additional Recommended Packages/triks

  • magick and EBImage - all kinds of image transformation, combination, animation, etc.
  • postr - making posters in R
  • profvis - profile the code: see which part of code runs longer
  • Adv. shiny tricks https://github.com/daattali/advanced-shiny
  • rMouse - control mouse
